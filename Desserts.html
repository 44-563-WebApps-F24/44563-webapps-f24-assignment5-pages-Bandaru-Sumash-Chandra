The letters use col-sm-2, which keeps them fixed in width on small screens, ensuring they stay in one line. In contrast, the dessert names use col-md, making them flexible and causing them to wrap when the screen size is reduced. This leads to the dessert names row breaking before the letters row as the browser width decreases.
